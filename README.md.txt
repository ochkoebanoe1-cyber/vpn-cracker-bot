# VPN Cracker Bot

## Быстрый старт

### На Render.com
1. Создай репозиторий на GitHub
2. Загрузи туда все файлы
3. На Render: New Web Service → выбери репозиторий
4. Добавь переменные: BOT_TOKEN, ADMIN_ID
5. Нажми Deploy

### На PythonAnywhere
1. Загрузи файлы через вкладку Files
2. В консоли: pip install --user -r requirements.txt
3. Запусти: python bot.py

### Локально
```bash
pip install -r requirements.txt
python bot.py
